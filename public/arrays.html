Arrays

(Methods implementation are below the Algorithm examples)

An array is a data structure for storing more than one data item that has a similar data type. The items of an array are allocated at adjacent memory locations. These memory locations are called elements of that array.
An array is a container of elements. Elements have a specific value and data type, like "ABC", TRUE or FALSE, etc. Each element also has its own index, which is used to access the element.
We are going to Discuss In Java programming language.

Why do we need arrays.

Arrays can be applied in various programming languages for various reasons that can include:
Arrays are best for storing multiple values in a single variable
Arrays are better at processing many values easily and quickly
Arrays are used to implement other abstract data structures like Queues and stacks
Sorting and searching the values is easier in arrays

Structure of an array
Basically the diagram above is trying to display the graphical representation of an array. Arrays contain items of the same datatype and these items are accessed through their array indices as shown above. Arrays usually have a fixed number of elements they can hold.
In java an empty array is declared by this statement:
int arr[]=new int[n];
Where n is the size of the array and int means it will contain integers.
But these arrays can also be declared with there elements an in the statement below:
int numbers[] = {300,200,100,50,0};

Operations on Arrays
There are 5 main Operations as named below:

Traverse −
Print all the elements in the array one by one. This can be effected using any kind of loop
Insertion −
Adds an element at the given index. eg. array[1] = 1;
Deletion −
Deletes an element at the given index. Can also Delete a whole array. eg. array.remove(new Integer(1)) // for the value 1 array.removeAt(1)//for index
Search −
Searches an element in the array using the given index or the value. Can also be effected using a loop.
Update −
Updates an element at the given index. For example array[0] = 1 // making any thing at index
